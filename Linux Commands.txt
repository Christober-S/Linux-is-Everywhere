pwd -> It will print the current working direcotory(Print Working Directory)

ls -> List files

whoami -> It will print the current used logged-in

ls {dir-name} -> It will list all files in the specified directory

cd {dir-name}-> Change directory

cd .. -> Undo 

mkdir {new-dir-name} -> make new directory with the specific directory name mentioned

touch {new-file-name} -> create new file with specific file name mentioned

rm {file-name} -> Remove specified file

rm -r {dir-name{ -> Remove specified directory

ls -R {dir-name} -> Shows recursively, current dir and all its sub dir contents

ls -a {dir-name} -> shows all including hidden files

ls -l {dir-name} -> Detailed output, metadata of dir contents

history - >It will display all history up to now

history {number-of-lines} -> It will display a specified number lines of history

clear -> It will clear all from the terminal

cat {file-name} -> It will display the content which are in the specified file

sudo passwd -l 'username' -> To change the password of a user

sudo userdel -r 'username' ->	To remove a newly created user

SSH username@ip-address or hostname  ->	login into a remote Linux machine using SSH

ip a -> Display all network interfaces and IP address

ps -> List running processes

uname -> Display system information

su -> Switch user to become another user
